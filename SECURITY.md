# Security Policy

## Supported Versions

This project is a static site and does not currently use a versioned release policy.

## Reporting a Vulnerability

Please do not open a public issue for a suspected security problem.

Instead, contact the repository owner directly and include:

- A short description of the issue
- Steps to reproduce it
- Any affected files or browser conditions
- Suggested mitigation if known

The maintainer can then validate the issue and decide on a fix path.
