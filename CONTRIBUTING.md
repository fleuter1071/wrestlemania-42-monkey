# Contributing

Thanks for contributing.

## Scope

This repository is a static HTML microsite. Keep contributions simple and focused.

## Before You Change Anything

1. Review [`README.md`](/C:/Users/dougs/.codex/wrestlemania-42-monkey/README.md) for project context.
2. Keep the page working as a plain static site.
3. Avoid adding frameworks or build tooling unless there is a clear reason.

## Preferred Changes

- Content updates
- Visual polish
- Accessibility improvements
- Better CTA behavior
- Cleaner code structure if the file grows too large

## Pull Request Notes

- Explain what changed and why
- Include screenshots for visual updates when helpful
- Call out any content placeholders that remain

## Local Check

Open `index.html` in a browser and verify:

- Images load correctly
- Sections still render on desktop and mobile widths
- Reveal animations still work
- The CTA behavior matches the intended flow
