# Repo Notes

## Purpose

This repository contains a single-file static event landing page concept.

## Working Rules

- Keep the site runnable as plain static HTML with no required build step unless explicitly needed.
- Prefer small, readable changes over introducing tooling.
- If the page grows meaningfully, split CSS and JavaScript out of `index.html`.

## Documentation Defaults

- Update `README.md` when the page structure, run instructions, or project purpose changes.
- Append meaningful completed work to `PROJECT_MEMORY.md`.
