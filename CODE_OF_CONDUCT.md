# Code of Conduct

## Our Standard

Please keep collaboration respectful, direct, and constructive.

Examples of expected behavior:

- Give clear, useful feedback
- Assume good intent
- Focus on the work, not the person
- Keep disagreements professional

Examples of unacceptable behavior:

- Personal attacks
- Harassment
- Discriminatory language
- Repeatedly derailing technical discussion

## Enforcement

The repository owner is responsible for interpreting and enforcing this standard.
